跨屏网（kuaping.com）发布phpcms后台跨屏移动版升级包，phpcms是国内知名的内容管理系统CMS，拥有大量的用户，随着移动互联网来临，越来越多的人不局限于在电脑前办公，phpcms传统的PC端后台已经无法满足随时随地管理的需要，就此跨屏网开发了phpcms跨屏版后台。

优势
-----

改动小，一共涉及到了2个文件 ，不改变原有代码，只是做了新增。

风格和原版PC保持统一
